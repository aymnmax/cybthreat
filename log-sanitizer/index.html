Built for SOC & IR Teams

Cybthreat Log Sanitizer v2 is a zero-dependency, fully client-side tool for stripping PII and sensitive identifiers from any log format before sharing with vendors, AI tools, or external analysts.

Every redaction is reversible. The token-to-original mapping stays in your browser — restore individual values or all at once without sending anything to a server.

12+ Detection Rules
IPs, emails, usernames, SIDs, UUIDs, MACs, paths, JWTs, API keys, and more
Smart JSON / KV Parsing
Field-aware redaction for structured logs — targets values by key name context
Reversible Tokens
Every [IP_1], [USER_2] maps back — restore selectively or all at once
Multi-Format Support
Windows Event JSON, Syslog, CEF, LEEF, Apache, Nginx, raw text — anything
Zero Data Egress
100% client-side. No server, no telemetry, no storage. GDPR-safe by design.
